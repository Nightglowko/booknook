Pill-shaped primary action button; use for any tap target that commits an action. Press shrinks gently; primary carries the candlelight-gold wash.

```jsx
<Button variant="primary" size="md" onClick={save}>Add to shelf</Button>
<Button variant="secondary" iconLeft={<Icon name="bookmark" />}>Want to read</Button>
<Button variant="ghost" size="sm">Skip</Button>
```

Variants: `primary` (gold wash), `secondary` (ink outline), `ghost` (text), `soft` (tinted fill), `danger` (oxblood). Sizes: `sm` `md` `lg`. Use `full` to stretch, `iconLeft`/`iconRight` for icons.
