The signature MoodShelf element: a handwritten, color-coded tag for the feelings a book left behind. Slightly rotated, like a real margin scribble.

```jsx
<EmotionTag emotion="heartbreak">destroyed me</EmotionTag>
<EmotionTag emotion="comfort" size="sm">like a warm bath</EmotionTag>
<EmotionTag emotion="longing" onRemove={remove}>aching</EmotionTag>
```

Emotions: joy, heartbreak, comfort, longing, eerie, tender, wonder, rage. Override hue with `color`. Sizes `sm` `md` `lg`.
