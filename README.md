# Book Nook local server

## Ouvrir le site dans VS Code

1. Ouvre le dossier du projet dans VS Code.
2. Lance la commande `Terminal: Run Task`.
3. Choisis `Serve MoodShelf`.
4. Ouvre ensuite dans ton navigateur :
   - `http://localhost:8000/`

## Fichier principal

- `outputs/index.html`

## Variante canonique

- `outputs/moodshelf-site/index.html`

## Arrêter le serveur

- Clique dans le terminal VS Code où le serveur tourne, puis fais `Ctrl+C`.
